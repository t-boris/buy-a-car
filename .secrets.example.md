# API Keys Template

⚠️ Copy this file to `secrets.md` and fill in your actual keys.

## Setup Instructions

1. Copy this template:
   ```bash
   cp .secrets.example.md secrets.md
   ```

2. Edit `secrets.md` and add your real API keys

3. Load keys into environment:
   ```bash
   source <(grep '^export' secrets.md)
   ```

## Export Commands Template

```bash
export GEMINI_API_KEY='your-gemini-api-key-here'
export GOOGLE_CSE_ID='your-google-cse-id-here'
export GOOGLE_API_KEY='your-google-search-api-key-here'
```

## Getting API Keys

### Gemini API
- Console: https://ai.google.dev/
- Click "Get API Key"
- Copy the key

### Google Custom Search
- Create Search Engine: https://programmablesearchengine.google.com/
- Get CSE ID from setup page
- Get API Key: https://console.cloud.google.com/apis/credentials
- Enable "Custom Search API"

## Verify Setup

```bash
# Check if keys are loaded
echo $GEMINI_API_KEY
echo $GOOGLE_CSE_ID
echo $GOOGLE_API_KEY

# Run local fetch
python run_local.py
```
